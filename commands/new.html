Forge

forge new

Creates a new project.

1: 

 new [--name <string>] [--dir <string>] [--template <string>] [--no-paket]

Options

--name <string>: Project name. If not provided, user will be prompted for it.

--dir <string>: Project directory. If not provided, user will be prompted for it.

--template <string>: Template name. If not provided, user will be prompted for it.

--no-paket: Forces no Paket installation, NuGet packages.config will be created.

--no-fake: Forces no FAKE installation, build scripts will not be created.
